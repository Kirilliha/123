# 123

## Лабораторная работа: Использование меню и диалоговых окон

Проекты на C# / .NET 8 Windows Forms. Каждый проект собирается командой
`dotnet build` из своей папки (требуется Windows для запуска).

### Проекты

- `wf341/` — Примеры 1 и 2. Простейший текстовый редактор:
  меню **файл** (открыть / сохранить), **вид** (шрифт / цвет),
  **справка**, **выход**. Используются `OpenFileDialog`,
  `SaveFileDialog`, `FontDialog`, `ColorDialog`, `MessageBox`.

- `sam1/` — Задание 1. Двойной щелчок по форме вызывает `ColorDialog`
  и меняет цвет формы.

- `sam2/` — Задание 2. Кнопка вызывает `OpenFileDialog` и загружает
  выбранное изображение в `PictureBox`.

- `sam3/` — Задание 3. Регистрационная форма: имя и город.
  Кнопки **Сохранить** / **Прочитать** работают с файлом `sam3.txt`.

- `sam4/` — Задание 4. Регистрационная форма: имя (`TextBox`),
  класс (`RadioButton`: 9 / 10 / 11), предмет (`ComboBox`: физика,
  математика, информатика). Кнопки **Сохранить** / **Прочитать**
  работают с файлом `reg.txt`.
